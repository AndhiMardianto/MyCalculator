My Calculator

Author: Andhi Mardianto

Introduction

My Calculator NVDA add-on is a calculator with multiple calculation methods and an improved user experience.

Available modes include standard, scientific, conversion, and more.

Press NVDA+Shift+M to open the calculator and enter your calculation. Press NVDA+Shift+H to hear the result of converting a Gregorian year to Hijri.

Each input produces a beep sound. You can also view calculation history. Press Enter, Tab, or "=" to move focus to the result. You can copy results to the clipboard. Press Esc to close the calculator.

Calculation Modes

Standard Mode

Uses standard mathematical rules (PEMDAS). Example: 3 + 5 * 2 = 13 (multiplication first).

Left to Right Mode

Calculates sequentially without operator precedence. Example: 3 + 5 * 2 = 16.

Scientific Mode

Includes sine, cosine, logarithms, and square root.

  • Sine and cosine use degrees (converted to radians automatically)
  • Supports natural logarithm (LN) and base-10 logarithm
  • Negative numbers are not supported for logarithms
  • Square root only works for non-negative numbers

Result formats:

  • High precision
  • Rounded (2 decimal places, use arrow keys to navigate)

Conversion Mode

Supports various conversions such as length, mass, and more.

Notes

  • Default mode is Standard when opening the calculator
  • You can use "X" for multiplication and ":" for division
  • Invalid input will be removed automatically
